# 🏗️ Terraform AWS EC2 Infrastructure

![Terraform CI](https://github.com/Ubani-Onu-C/terraform-aws-ec2/actions/workflows/terraform.yml/badge.svg)

## 📌 Project Overview
Infrastructure as Code (IaC) project that provisions a production-ready AWS EC2 
web server using Terraform — fully validated through an automated CI/CD pipeline 
on every code push.

## 🛠️ Technologies Used
- **Terraform 1.10** — Infrastructure as Code
- **AWS EC2** — Cloud virtual machine
- **AWS Security Groups** — Network access control
- **GitHub Actions** — Automated validation pipeline
- **Linux/Bash** — Server bootstrap scripting
